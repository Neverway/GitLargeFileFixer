 # 😺🛠Git Large File Fixer Storage (LFFS)

## About

Dealing with project files that suddenly became too large and requiring migration to LFS is a headache! I got tired of running all of those commands for every file, so I created this web ui to make it easier!


## Requirements
### Program Requirements
- Python = 3.11 (Others may work, this is the only one I tested it on though)
- nicegui ~= 3.8.0
- Git
- Git LFS

### System Requirements

- **OS:** Linux, preferably
- **Memory:** Whatever ram you can get ur hands on
- **Storage:** 150MB free space

## Installation

### Windows/Linux/Mac probably too Idk
Download the archive for the latest release and extract it!
-  😸💽 [Latest Release](https://github.com/Neverway/GitLargeFileFixer/releases)
## Usage
1.) In the terminal or command prompt (or whatever is going on on mac) navigate to the folder where you extracted LFFS. (Replace `/path/to/GitLFFS` with the actual path to your folder for the program)
### Linux/Windows
`cd /path/to/GitLFFS`
### Mac
`Sacrifice first born probably`

2.) Ensure you have installed all the requirements
### Linux/Windows
`pip install -r requirements.txt --break-system-packages`
### Mac
`Sacrifice a goat or two to a two headed serpent (Store bought is fine)`

3.) Run `lfsMigrateGUI.py` and it should open up a browser page to `http://127.0.0.1:8080/`
### Linux/Windows
`python3 ./lfsMigrateGUI.py`
